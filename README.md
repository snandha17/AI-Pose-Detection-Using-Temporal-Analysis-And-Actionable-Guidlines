# AI-Pose-Detection-Using-Temporal-Analysis-And-Actionable-Guidlines
PoseGuru is a real-time Human Pose Correction System designed to analyze biomechanics, track movement sequences, and provide actionable on-screen feedback. While traditional pose estimators only analyze static frames, PoseGuru leverages Temporal Analysis to understand the continuous flow of an exercise, ensuring perfect form from start to finish.
# PoseGuru

## Features
- Real-time pose detection
- Posture correction
- Exercise tracking
- Streamlit interface
